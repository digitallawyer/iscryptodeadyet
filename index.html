The one number. The navy line sums six Google search phrasings — bitcoin dead, crypto dead, bitcoin is dead, crypto is dead, is bitcoin dead, is crypto dead — from one worldwide Trends query, rescaled so the all-time peak equals 100.

How to read it. Bitcoin price uses the left axis — toggle Linear or Log above. Search volume uses the right axis. The current bitcoin price updates live; search data runs through May 2026.

Sources. Bitcoin: monthly closes from CryptoCompare, current month from CoinGecko. Search: Google Trends (worldwide, monthly) — download the CSV or re-run the query.
